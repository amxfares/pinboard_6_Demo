## 📌 Pinboard — Pinterest-Inspired UI Clone

A fully responsive image discovery and inspiration board built from scratch
using pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

Created as a frontend portfolio demonstration showcasing UI/UX skills,
clean code structure, and attention to design detail.

---

### ✨ Features

- 🧱 Masonry grid layout with dynamic multi-column flow
- 🔍 Live search — filters by title, author, category, or description
- 🗂️ Category tabs — Nature, Architecture, Food, Fashion, Travel, Art & more
- 💾 Save / Unsave pins with persistent state during session
- 🖼️ Pin detail modal with smooth open/close animations
- 🔗 Share button with clipboard copy
- 📊 Scroll progress indicator
- 📱 Fully responsive down to mobile (2-column grid)
- ⚡ Zero dependencies — vanilla HTML, CSS & JS only

---

### 🛠️ Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| Structure  | HTML5                   |
| Styling    | CSS3 (Custom Properties, Flexbox, CSS Columns) |
| Logic      | Vanilla JavaScript (ES6+) |
| Fonts      | Google Fonts (Playfair Display + DM Sans) |
| Images     | Unsplash API (static URLs) |
| Avatars    | UI Avatars API          |
